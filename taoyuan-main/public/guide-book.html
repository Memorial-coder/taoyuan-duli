2. 开始前与存档

  • 账号状态：建议先登录再开档。当前主菜单文案已经明确提醒，未登录直接开始旅程时，存档无法保存。
  • 存档方式：支持本地存储和服务端持久化。认真长期玩，优先服务端持久化。
  • 保存与保存并返回：都依赖当前活跃存档槽位，适合在关键节点和准备退出前使用。
  • 导入 / 导出 / WebDAV：更像额外同步与备份手段，不替代你理解当前存档模式。
  • 在线功能：交流大厅、邮箱、额度兑换等都默认围绕登录状态和当前服务端存档来工作。

3. 田庄与前七天

田庄现在更适合谁开局建议
桃源田庄完全新手、种田保底流当前版本默认推荐，最容易学会主线、商圈、告示板与稳定现金流。
草甸田庄想更早体验养殖的人前期节奏舒服，但会更早进入牧场与动物维护。
溪流田庄想靠钓鱼起现金流的人适合你愿意围绕天气和时间行动。
竹林 / 山丘 / 荒野采集、矿洞、战斗、成长流更适合你已经知道自己想玩什么时再选。

当前版本前七天最重要的不是把所有系统摸一遍，而是先跑出一条最小可持续循环：主线不断、收入不断、知道下一步去哪个页面。

4. 当前版本核心循环

看任务页

主线、日常委托、特殊订单、限时活动，会给你最直接的下一步方向。

看目标规划

主题周、市场轮换、今日目标、连周进度，决定这一周最值得承接什么。

去对应系统出货

农场、商圈、钓鱼、矿洞、牧场、加工，会把“方向”变成真正的资源和钱。

把收益回流

工具、仓储、设施、预算、高价投入，决定你下周能承接更高一级的循环。

周结算与活动

周目标结算、奖励池、周赛、馆务和赛季，会把你这一周的结果重新放大。

再开新系统

基础线稳定后，鱼塘、育种、博物馆、公会、瀚海、钱袋预算才会真正有深度。

迷路时的默认顺序：先看任务页，再看顶部目标规划和各页引导面板。

5. 任务板 / 主题周 / 周结算

主线任务

告诉你当前阶段要解锁什么、该碰哪些系统。开荒时最重要。

日常委托

提供短期钱、材料、好感和路线提示，是把村民关系与经营节奏接起来的稳入口。

特殊订单

现在会承接育种、鱼塘、家庭 / 仙缘等方向，不只是“多给一点钱”。

主题周 / 活动编排

会影响商圈推荐、任务风向、周重点目标，以及多个长线系统承接。

村庄建设线路

任务页已经会直接告诉你哪些建设能增强任务容量、收益或相关入口。

周结算

周目标、奖励池、连周进度、预算重置与各种周赛，都会在这里形成闭环。

6. 农场、牧场、加工

  • 农场 依然是最稳保底现金流，前期别铺太满，重点是稳定浇完和稳定续种。
  • 牧场 不再只是摆设，动物线会和日常经营、加工链、长期收藏一起推进。
  • 烹饪与加工 的作用更大了：既是恢复和礼物，也会是任务材料和高价值承接物。
  • 仓储 比早期版本更重要，因为任务、加工、育种、鱼塘都会持续吃材料。
回新手教程：什么时候开第二条线返回顶部

7. 钓鱼与鱼塘

钓鱼

仍然是前期很好的副现金流，尤其适合溪流田庄，也能顺手为后续鱼塘提供样本。

鱼塘

  • 喂食、水质改良、病鱼治疗、每日收获
  • 周赛报名与结算
  • 高阶养护，例如观赏饲料和高级净水剂
  • 展示池 / 观赏缸，对周赛和博物馆展陈都有反向加分
  • 经营评级 / 资格快照，帮助你判断哪些样鱼能走更高价值路线

所以鱼塘更适合在你已经稳定钓到鱼、也有余力维护水质和产出之后再认真开。

回新手教程：第二条线时机返回顶部

8. 矿洞与公会

  • 矿洞 仍然是矿石、稀有材料、装备成长和后续解锁的重要来源。
  • 瀚海 的开通条件之一就是击败矿洞第 120 层 BOSS。
  • 公会 现在已经有赛季阶段、荣誉档位、奖励池、讨伐、捐献、商店、怪物图鉴，以及和主题周、任务板、成就的联动。

如果你走的是战斗或下矿成长流，公会会成为中后期非常重要的承接页。

回新手教程：第二条线时机返回顶部

9. 育种

育种台与种子箱

先用筛选和排序缩小范围，再决定是补现有谱系，还是冲新杂交品种。

研究与大师化

研究等级会影响识别范围、速育门槛、失败损耗和谱系深度。

图鉴与谱系认证

当前版本已支持图鉴、世代、综合分和认证供货线的长期记录。

育种周赛 / 品鉴会

不仅能报名，还有周切换刷新与结算，已经是一条稳定周循环内容。

失败回收与物流补材

高代失败不再只是简单亏掉，会回收到研究、认证和储运相关材料。

育种规划器

会汇总本周订单、下周主题周、图鉴缺口和推荐亲本，帮助你决定这一周押哪条线。

育种更像“长期经营中的实验室”，不是新手第一周要硬开的生产页。

10. 村民、婚姻、仙灵

  • 村民关系 不是纯装饰，很多委托、剧情和长期陪伴感都要靠关系线承接。
  • 婚姻与家庭 当前可婚 NPC 已支持同性婚姻；同性之间也保留知己线。
  • 仙灵 已经是单独长线：传闻 → 惊鸿一瞥 → 初次相遇 → 愿意往来，再通过互动、供奉、求缘、结缘解锁长期能力。

仙灵更适合中后期慢慢推进，不建议开局就把全部注意力放在找仙灵上。

回新手教程：快速查询入口返回顶部

11. 博物馆

  • 基础层：古物、化石和特殊收藏的捐赠。
  • 馆务层：展陈等级、展示评分、访客热度、祠堂主题、馆区推进。
  • 委托层：学者委托已经是明确可接、可领奖的中线内容。
  • 联动层：会反向影响告示板、推荐动作、支持 NPC 与长期经营焦点。

如果你是收藏流或展示流玩家，博物馆现在已经值得单独当一条经营主线对待。

回新手教程：快速查询入口返回顶部

12. 瀚海

  • 需要击败矿洞第 120 层 BOSS 并支付修路费，明显属于中后期解锁内容。
  • 当前包含驿站商店、遗迹勘探、瀚海赌坊、商路投资、合同、遗迹套组、轮换货架。
  • 还会和主题周、村庄建设、目录货架一起联动。

瀚海现在更像高阶段的“商路经营与特殊区域玩法”，不是单纯的额外地图。

回新手教程：长线系统何时开返回顶部

13. 钱袋、预算与高价投入

后期经济观测

会告诉你当前经营分层、风险提示和推荐资金去向。

豪华消费路线

当前商圈会按你的路线给出更契合的高价货架和本周精选。

周预算系统

每周可以为商路、展馆、学舍等频道投入预算，当周生效、下周重选。

资源券 / 凭证

会跟着周目标、预算和长期经营一起运转，是当前版本的钱袋特色内容。

当前版本不是“赚到钱就结束”，而是“把钱继续投进更高级的经营结构里”。

14. 在线功能

  • 登录 / 注册：决定你能否长期保存、读取账号相关进度。
  • 服务端持久化：长期游玩优先。
  • 交流大厅：发帖、回复、求助、看其他玩家经验。
  • 邮箱：接收公告、奖励和活动内容。
  • AI 助手：适合问玩法、路线和卡点，但不会提供隐藏后台规则。
  • 额度兑换：需要切到服务端持久化并载入目标存档后才能继续。

15. 常见问题 FAQ

Q1：第一次玩先看哪个页面最有效？

先看新手教程，再进游戏看任务页 / 告示板。

Q2：第一周最重要的不是赚钱吗？

赚钱重要，但当前版本更看重你有没有把主线、告示板和稳定收益线跑顺。

Q3：为什么现在总强调主题周？

因为它已经会影响商圈推荐、目标规划、特殊订单和多个长线系统承接。

Q4：鱼塘和育种哪个更早开？

通常鱼塘比育种更早，育种明显更偏中后期。

Q5：公会和博物馆只是兴趣内容吗？

不是，它们都已经有各自的成长、奖励、结算和跨系统影响。

Q6：瀚海什么时候值得开？

当你能稳定下矿、打过 120 层 BOSS、且经济不再吃紧时再开体验最好。

Q7：为什么额度兑换有时不能用？

因为它要求你切到服务端持久化，并且已经载入目标服务端存档。

Q8：主菜单百科和游戏内百科有什么区别？

主菜单百科看系统结构，游戏内百科查具体机制、来源、用途和送礼。
