1. 游戏定位与核心玩法

桃源乡 是一款像素风田园经营游戏，但它绝不只是“种地”。

随着流程推进，你会逐渐接触到：

  • 农耕：开地、播种、浇水、施肥、收获、扩建
  • 养殖：鸡舍、畜棚、动物照顾、产物经营
  • 钓鱼：不同地点、天气、时间、季节影响鱼种和收益
  • 挖矿与战斗：矿洞探索、怪物、BOSS、装备、掉落和成长
  • 烹饪与加工：把原料做成恢复品、增益品或高价值商品
  • 社交：和村民交流、送礼、推进关系、结婚成家
  • 收集：公会、博物馆、成就、隐藏 NPC、特殊剧情

一句话总结：

桃源乡是一款“农场经营 + 资源探索 + 角色成长 + 村落生活”混合在一起的长期经营游戏。

如果你喜欢慢慢建设自己的农场和生活节奏，它会越玩越有意思。

2. 开局前必须知道的 10 件事

  1. 体力很重要：绝大多数行动都会消耗体力。
  2. 时间是会流逝的：每天能做的事有限，睡得太晚会有惩罚。
  3. 前期最缺的是钱、体力、背包空间和仓储能力。
  4. 不要平均发展所有系统，先养出一条稳定收益线。
  5. 换季前几天一定要注意成熟时间，不然作物会白白枯萎。
  6. 重要节点要手动保存，尤其是下矿、购物、换季前后。
  7. 不要把所有材料都卖光，很多稀有材料中后期会非常缺。
  8. 村民社交不是装饰，它关系到剧情、婚姻、奖励和隐藏内容。
  9. 不同田庄开局差异很大，选图会影响你前期节奏。
  10. 桃源乡现在和抽奖系统联动，你可以保存后返回，还能兑换额度。

新手最稳妥的原则：

先种田保底，再用剩余体力去钓鱼、采集或下矿。

3. 如何开始一局新游戏

进入桃源乡主菜单后，你会看到：

  • 本地存储 / 服务端持久化切换
  • 新的旅程
  • 返回抽奖系统（后台开启时显示）
  • 关于游戏（后台开启时显示）
  • 存档列表
  • 导入存档

新游戏开始流程

  1. 点击 新的旅程
  2. 阅读并同意隐私协议
  3. 输入角色名字（最多 4 个字）
  4. 选择角色性别
  5. 进入田庄选择界面
  6. 选择一个田庄类型
  7. 确认后正式进入游戏

系统会自动做什么

开新局时系统会自动：

  • 分配一个空闲存档槽位
  • 清空上一局残留状态
  • 写入角色身份
  • 创建初始农场
  • 发放开局基础物资（例如种子）
  • 初始化主线任务
  • 若开启新手提示，会写入第一条引导日志

也就是说：

“新的旅程”一旦进入，就是一局已经绑定了存档槽位的新游戏。

5. 主界面完整说明

5.1 顶部状态栏

顶部会显示：

  • 地点名称（桃源乡）

  • 当前年份

状态栏下方会有一个休息按钮，文字可能是：

  • 作坊
  • 育种

  • 地图按钮

  • 日志按钮

6. 顶部状态栏与新增按钮详解

现在顶部状态栏已经加入了两个非常关键的新功能：

6.1 顶栏“保存”

作用：

  • 把当前进度保存到当前活跃存档槽位

推荐使用时机：

  • 大丰收之后
  • 下矿之前
  • 大额消费之前或之后
  • 准备睡觉之前

6.2 顶栏“保存并返回”

作用：

  1. 先保存当前进度
  2. 保存成功后返回抽奖系统

适合场景：

  • 你准备结束本次桃源乡游玩
  • 想切回 Lucky 抽奖系统前先锁定进度

6.3 两个按钮的失败保护

如果发生以下情况：

  • 没有活跃存档槽位
  • 保存失败

系统会：

  • 给出提示
  • 不会误跳转

因此“保存并返回”是安全的。

7. 时间、季节、天气、体力与睡觉机制

7.1 时间结构

游戏内时间分为：

  • 年
  • 季（春 / 夏 / 秋 / 冬）
  • 天（每季 28 天）
  • 小时

一天通常从 6:00 开始。

7.2 晚睡与昏倒惩罚

如果你睡得太晚：

  • 第二天体力恢复会打折
  • 如果太晚或体力归零，可能直接昏倒
  • 昏倒会损失一部分铜钱

所以不要把“把今天榨干”当作长期打法。

7.3 天气影响

天气包括：

  • 晴
  • 雨
  • 雷雨
  • 雪
  • 大风
  • 绿雨

它们会影响：

  • 钓鱼收益
  • 采集概率
  • 部分地点表现
  • 一些田庄特效

7.4 体力机制

以下行为会消耗体力：

  • 锄地
  • 浇水
  • 采集
  • 钓鱼
  • 下矿
  • 战斗
  • 一些特殊操作

恢复体力的方法：

  • 睡觉
  • 吃料理
  • 使用恢复型物品
  • 依靠某些加成效果

8. 存档系统超详细说明

桃源乡现在支持：

  • 本地存储
  • 服务端持久化
  • 顶栏保存
  • 顶栏保存并返回
  • 存档管理页显式保存
  • 导入存档
  • 导出存档
  • WebDAV 上传 / 下载

8.1 存档槽位

默认有 3 个槽位。

每个槽位会显示：

  • 槽位编号
  • 玩家名字
  • 当前年份 / 季节 / 天数

8.2 什么是活跃存档槽位

活跃槽位就是你当前正在玩的那个槽位：

  • 新游戏时自动分配
  • 加载旧存档时自动切换

顶部“保存 / 保存并返回”都依赖它。

8.3 本地存储

特点：

  • 保存在当前浏览器里
  • 清缓存后可能丢失
  • 换设备不会自动同步

适合：

  • 临时试玩
  • 单设备玩家

8.4 服务端持久化

特点：

  • 存档和登录账号绑定
  • 更适合长期游玩
  • 换设备也能继续读取

推荐长期用户使用。

8.5 导入 / 导出存档

导出

用途：

  • 备份进度
  • 迁移设备
  • 防止误删或缓存清空

导入

用途：

  • 恢复备份
  • 把别处的存档移回来

如果没有空槽位，系统会先提示你删除旧存档。

8.6 存档管理入口

在游戏内：

  1. 打开 设置
  2. 点击底部 存档管理

里面可以：

  • 查看槽位
  • 加载存档
  • 删除存档
  • 导入 / 导出
  • 切换本地 / 服务端模式
  • 显式保存当前进度
  • WebDAV 上传 / 下载

9. 新手第一天到第一周建议路线

第一天建议做什么

  1. 先看顶部状态栏：时间、天气、体力
  2. 去农场开少量地
  3. 把送的种子种下
  4. 浇水
  5. 还有体力就去采集 / 钓鱼简单试试
  6. 睡前手动保存一次

前三天目标

  • 学会播种、浇水、收获
  • 熟悉任务、日志、背包、商店
  • 建立第一条赚钱路线

第一周目标

  • 保持每天稳定收益
  • 升一个关键项目（背包 / 工具 / 仓储）
  • 不要乱花钱导致断粮
  • 再慢慢接触第二条副收入路线

新手最稳路线

路线 A：种田稳健流

  • 优先选成熟快、回本快的作物
  • 每天稳定浇水
  • 逐步扩大面积
  • 中期再接加工链

路线 B：钓鱼现金流

  • 适合溪流田庄
  • 前期赚钱手感很好
  • 但更依赖操作与时机

路线 C：采集 / 挖矿成长流

  • 适合竹林 / 山丘 / 荒野
  • 资源增长更快
  • 更快进入工具、装备、加工系统

10. 农场系统详解

农场是最稳定、最基础的经济来源。

10.1 标准循环

  1. 开地
  2. 播种
  3. 浇水
  4. 等待成熟
  5. 收获
  6. 卖出或加工

10.2 前期不要铺太满

很多新手的错误是：

  • 第一天下体力全拿去开地
  • 第二天浇水就把自己累空

建议：

只种你每天能稳定浇得完的数量。

10.3 作物选择原则

  • 看当前季节能否种植
  • 看成熟时间
  • 看售价是否划算

19. 隐藏 NPC、缘分与特殊发现

除了普通村民，游戏里还有隐藏角色、仙灵、缘分与发现链。

这套系统通常不是一次性触发，而是按阶段慢慢推进：

  1. 传闻：先听到传说、拿到线索，或在日志里看到异常提示。
  2. 惊鸿一瞥：在合适的地点、时间、天气或季节里，见到短暂异象。
  3. 初次相遇：准备关键物品，或满足技能 / 主线 / 好感等前置后正式见面。
  4. 愿意往来：达到最终解锁条件后，仙灵会常驻在仙灵面板中。

常见触发条件包括：

  • 特定地点
  • 特定时间
  • 特定天气或季节
  • 特定主线进度
  • 特定技能等级
  • 持有关键道具
  • 与某位村民达到指定好感

发现之后，玩法会继续分成两层：

  • 互动 / 供奉：用于提高缘分值，逐步解锁仙缘能力。
  • 求缘 / 结缘：达到门槛并备好信物后，可建立更深层羁绊，获得长期加成。

可以把仙灵系统理解成：先找到仙灵，再靠互动和供奉养缘分，最后用信物完成求缘与结缘。

如果你不知道该从哪里下手，最稳妥的思路是：

  • 先把农耕 / 采集 / 钓鱼 / 挖矿的基础等级练起来
  • 多去竹林、瀑布、矿洞、村庄这些特殊地点
  • 留意满月、雨夜、深夜、清晨等特殊时段
  • 看到异常日志后，优先围绕那条线索继续尝试

这部分更适合中后期慢慢推进，不必开局强求；但如果你想专门追仙灵，记得优先看仙灵页里的传闻提示和下一步建议。

26. 常见问题 FAQ

Q1：为什么顶部保存按钮提示没有活跃存档？

A：你要先新开一局或先加载一个已有存档。

Q2：保存并返回安全吗？

A：安全。逻辑是先保存，保存成功才返回，失败不会跳走。

Q3：为什么我看不到返回抽奖系统 / 关于游戏？

A：管理员可能在后台关闭了这些入口。

Q4：本地存储和服务端持久化能切换吗？

A：可以，但它们是两套数据来源，切换后请先确认当前读取到的是哪一套存档。

Q5：导出存档有必要吗？

A：非常有必要，尤其是长期游玩和重要节点前后。

Q6：额度兑换是不是实时到账？

A：正常情况下是即时处理的，成功后会同步刷新显示。

Q7：顶栏保存和存档管理里的保存哪个好？

A：本质一样。顶栏更快，存档管理更适合看清楚槽位再操作。

Q8：前期优先升级什么？

A：多数情况下优先：

  1. 稳定收入来源
  2. 背包 / 工具
  3. 仓库
  4. 加工链
  5. 动物 / 鱼塘 / 深层探索
