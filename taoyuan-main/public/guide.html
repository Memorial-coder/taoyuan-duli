1. 开始前确认

账号

建议先登录再开档。当前主菜单已经明确提醒：未登录直接开始旅程时，存档无法保存。

  • 想长期玩：先注册 / 登录
  • 想用交流大厅、邮箱、额度兑换：必须先登录
  • 想减少折腾：开档前先处理完

存档方式

支持 本地存储 和 服务端持久化。认真长期玩，优先服务端持久化。

  • 本地存储：偏当前设备
  • 服务端持久化：按账号读取，更适合长期游玩
  • 额度兑换：还要求载入目标服务端存档

这些入口先知道，但不用一开始全研究

交流大厅、邮箱、AI 助手、关于游戏这些入口都已经在主菜单里了，但第一次开局依然先把主线、告示板和基本经营循环跑起来。

2. 先记住这 5 条

  1. 第一次玩优先选桃源田庄。 想更早碰养殖再选草甸田庄。
  2. 别平均碰所有系统。 当前版本系统很多，但前期还是先养出一条稳定收益线。
  3. 不知道做什么就先看任务页。 导航里叫告示板，里面有主线、委托、特殊订单和活动窗口。
  4. 开始中期后要学会看主题周和市场。 当前版本已经不是闷头刷一个点就够了。
  5. 保存别拖。 下矿前、换季前、大额消费前后、准备退出前都值得手动保存。
任务给方向，种田保底，副线补钱，周目标承接，长线系统慢慢开。

3. 六种田庄怎么选

桃源田庄

当前版本最稳开局。最容易学会主线、种田、商圈、告示板这些核心循环。

草甸田庄

想更早体验牧场和动物线就选它，但你会更早进入喂养和产物维护。

溪流田庄

适合想靠钓鱼起现金流的人。赚钱手感好，但更依赖时间、天气和效率。

竹林 / 山丘 / 荒野

更偏采集、矿洞、战斗、成长流。适合你已经知道自己想玩什么时再选。

4. 第一天到第一周

第一天

  • 确认账号和存档方式，再正式开局
  • 只开你第二天能稳定浇完的地
  • 先看主线和任务页
  • 认一下商圈、背包、角色、告示板在哪

前 2 到 3 天

  • 先养出第一条稳定现金流
  • 体力有余就用钓鱼或采集补钱
  • 开始做简单委托
  • 第一笔大钱优先给背包、仓储、工具或种子

第一周

  • 保持主线不停
  • 开始理解主题周、市场和今日目标
  • 最多补一条副线
  • 别同时硬开鱼塘、育种、瀚海

第一周成功标准

不是“赚到最多钱”，而是你已经同时做到三件事：有稳定收入、主线没断、知道下一步该去哪一个页面。

5. 当前版本核心循环

1. 看任务页 / 告示板

主线、日常委托、特殊订单、限时活动会告诉你今天最值的方向。

2. 看目标规划

主题周、市场轮换、今日目标、活动，会决定这周最值得承接什么。

3. 去对应系统出货

农场、商圈、钓鱼、矿洞、牧场、加工，把“方向”变成资源和钱。

4. 把收益回流

工具、仓储、设施、预算和高价投入，决定你下周能承接多高一级的循环。

5. 周结算后开新线

基础线稳定后，再去接鱼塘、育种、博物馆、公会、瀚海这些长线系统。

迷路时的默认答案

先看任务页，再看顶部目标规划和引导面板。

6. 什么时候开第二条线

钓鱼 / 采集

最适合第一条副线。前期缺现金流、缺任务材料时都很实用。

矿洞

开始明显缺矿石、工具材料和装备成长时就能开，但不建议第一周完全压重心。

牧场

草甸田庄可以更早碰；其他田庄等浇地和任务节奏顺手后再接进来。

鱼塘

现在鱼塘有周赛、高阶养护、展示池和博物馆联动，更适合中期再认真开。

育种

明显是中后期系统。研究、认证、周赛、规划器都要求你先把基础生产链跑顺。

博物馆 / 公会 / 瀚海 / 钱袋

它们都已经是完整长线内容，适合你能稳定赚钱和推进周目标后再投入。

7. 常见卡点

我不知道今天该做什么

先开任务页 / 告示板，再看目标规划和引导面板。当前版本默认就是让它们告诉你下一步。

我总觉得钱不够

先确保种田保底没断，再用钓鱼 / 采集补钱，不要一上来就把钱砸进多个长线系统。

材料总是不够用

现在任务、加工、育种、鱼塘都会吃材料，别把所有东西都卖光。

我是不是该立刻去看百科

不用先整页通读。新手页先看完，碰到具体问题时再去百科按主题查更有效。

9. 新手 FAQ

Q1：我不登录也能直接玩吗？

A：能进入流程，但当前主菜单已经明确提示：未登录直接开始旅程时，存档无法保存。

Q2：第一周最重要的成果是什么？

A：有稳定收入、主线没断、知道下一步该去哪一个页面。

Q3：什么时候开始碰鱼塘和育种？

A：鱼塘通常是中期；育种更偏中后期。两者现在都带周赛、评级和联动，不适合第一周硬开。

Q4：我需要现在就把百科全书整页看完吗？

A：不需要。先把这页看完，再按问题跳去百科查具体机制，效率更高。
