1. 新手先看这 5 条

  1. 推荐开局：第一次玩优先选 桃源田庄，想早点体验养殖可以选 草甸田庄。
  2. 第一天别开太多地：只种你第二天还能稳定浇完的数量。
  3. 前期赚钱最稳：种田保底，钓鱼/采集补现金流。
  4. 不要太晚睡：熬夜和昏倒都会拖慢节奏，前期尤其伤。
  5. 关键节点手动保存：下矿前、换季前、大额消费前后、准备退出前，都值得保存一次。

2. 如何开始一局新游戏

进入主菜单后，先点 新的旅程，然后按下面顺序走：

  1. 阅读并同意隐私协议
  2. 输入角色名字（最多 4 个字）
  3. 选择角色性别
  4. 选择田庄类型
  5. 确认后进入游戏

新手最容易忽略的点

“新的旅程”一旦进入，就是一局已经绑定存档槽位的新游戏。 所以后面如果中断，别忘了手动保存。

3. 开局田庄怎么选

六种田庄都能玩，但第一次上手没必要追求最特殊的路线。先选好理解、失误成本低的。

最推荐：桃源田庄

适合完全新手。节奏稳定，农场基础最好，最容易学会桃源乡的核心循环。

看桃源田庄完整说明

次推荐：草甸田庄

适合想早点碰养殖的人。前期经济更舒服，但你会更早接触动物照顾与建筑管理。

看养殖玩法详解

想快点赚钱：溪流田庄

适合喜欢钓鱼手感和现金流的人，收益不错，但更依赖操作和时机。

看钓鱼与鱼塘详解

想快点成长：竹林 / 山丘 / 荒野

更偏采集、挖矿、战斗路线，适合第二轮或已经有目标的玩家。

看采集与矿洞详解

一句话建议

第一次玩：优先桃源；想要更早体验养殖：选草甸。

4. 第一天 / 前三天 / 第一周照着做

第一天

  1. 先看顶部状态栏，确认时间、天气、体力。
  2. 回农场开少量地，不要一口气铺满。
  3. 把送的种子种下并浇水。
  4. 有剩余体力就去简单采集或钓鱼。
  5. 准备休息前手动保存一次。

前三天

  • 熟悉播种、浇水、收获、背包、商店、任务、日志。
  • 建立第一条稳定赚钱路线，不要每个系统都碰一遍。
  • 开始意识到体力和背包空间都很珍贵。

第一周

  • 保持每天有稳定收益。
  • 优先升级一个关键点：背包 / 工具 / 仓储。
  • 再慢慢接第二条副收入路线。

这一段最重要的不是“做很多”，而是“做稳定”

很多新手第一周最大问题不是不会赚钱，而是所有系统都浅尝辄止，结果每条线都没起来。

5. 前期赚钱的 3 条路线

路线 A：种田稳健流

最适合新手，回报稳定，最容易理解节奏。

  • 优先成熟快、回本快的作物
  • 每天稳定浇水
  • 逐步扩大面积
查看农场系统详解

路线 B：钓鱼现金流

来钱快，适合补前期现金，但更吃手感和时机。

  • 适合溪流田庄
  • 不依赖作物成熟时间
  • 天气、地点、时间都会影响收益
查看钓鱼与鱼塘详解

路线 C：采集 / 挖矿成长流

更偏成长和资源积累，适合目标明确的玩家。

  • 工具、装备、材料成长更快
  • 能更早接入加工链
  • 风险也会更高
查看采集与矿洞详解

新手默认搭配

种田保底 + 钓鱼/采集补现金流，这是最稳、最不容易翻车的组合。

6. 新手最容易踩的坑

误区 1：第一周什么都想做

结果往往是每条线都半吊子。先养出一条稳定收入线，其他系统再慢慢补。

误区 2：不手动保存

尤其下矿、换季、购物、大丰收前后，忘记保存最容易让人后悔。

误区 3：换季前乱种

成熟时间对不上季节，作物会枯萎，属于最常见的直接亏损。

误区 4：什么都卖

很多材料中后期会非常缺，尤其矿物、高品质材料和特殊掉落。

误区 5：血量和体力见底还硬冲

矿洞里尤其容易翻车。感觉不稳时，保住今天收益比硬赌更重要。

误区 6：把百科当成开局必读

你不需要一次看完全部系统。新手页先看完，再按问题跳去百科查即可。

8. 新手 FAQ

Q1：为什么保存按钮提示没有活跃存档？

A：你需要先新开一局，或先加载一个已有存档。顶栏保存依赖当前活跃槽位。

看活跃存档槽位解释

Q2：保存并返回安全吗？

A：安全。逻辑是先保存，保存成功才跳转；失败不会误跳走。

看保存并返回详解

Q3：前期优先升级什么？

A：通常优先：稳定收入来源 → 背包 / 工具 → 仓储 → 加工链。
